# RCF Design Project Deliverables

Client-facing system design reports for RCF USA installed sound projects, hosted via GitHub Pages.

Each project gets a dedicated subfolder containing a single self-contained HTML page with everything the client needs:
- System overview and coordinate reference
- Design walkthrough video link
- SPL coverage views (isometric, side, front, top)
- Cluster summary and per-cluster detail (position, aim, splay, rigging)
- Amplifier channel allocation
- Bill of materials (installer view, no pricing)

## Agent Workflow

Reports are generated by the Craft Agent workflow:

1. Fetch project metadata from NocoDB + Streak
2. Parse EASE CSV export (cluster positions, splay, amp assignments)
3. Query NocoDB BOM — inject rigging notes by speaker model + hang type
4. Populate HTML template
5. Commit to this repo → GitHub Pages publishes automatically
6. Pages URL written back to NocoDB project record (RCF Project Deliverables URL)
